# tinder_likes_data
Gathering data of likes on [Tinder](https://tinder.com/) within the past 7 days

## Non-Python Requirements
- Tinder Platinum.
- Google Chrome running on localhost. See [this video](https://youtu.be/FVumnHy5Tzo) for details.
